# Football D20 Sandbox

Sandbox matematic pentru testarea mecanicilor D20 pentru jocul de fotbal pe tablă.

## Local

```bash
npm install
npm run dev
```

Deschide:

```text
http://localhost:3000
```

## Build

```bash
npm run build
```

## Vercel

Proiect Next.js standard. În Vercel:

- Framework Preset: Next.js
- Build Command: `npm run build`
- Install Command: `npm install`
- Output Directory: implicit / gol
